# Fluxo de Projetos

## Fluxo Geral

```text
Upload de letra/beat
    ↓
Geração de hash
    ↓
Análise IA
    ↓
Busca vetorial
    ↓
Recomendação
    ↓
Match confirmado
    ↓
Chat privado
    ↓
Pré-contrato
    ↓
Validação coletiva
    ↓
Hash oficial
    ↓
Videochamada
    ↓
Projeto colaborativo
    ↓
Entregas
    ↓
Definição de %
    ↓
Conclusão oficial
```

# Match Inicial

A IA identifica compatibilidade entre usuários.

---

# Validação

Os participantes possuem:

```text
24 horas
```

para validar o projeto.

---

# Substituição Automática

Se houver inatividade:

- o sistema procura substitutos,
- mantém o fluxo ativo.

---

# Projeto Colaborativo

Após validação:

- timeline é criada,
- tarefas são organizadas,
- entregas são monitoradas.

---

# Finalização

A conclusão exige:

- validação coletiva,
- aprovação financeira,
- confirmação das entregas.
