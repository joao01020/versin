# Contributing to Versin

Obrigado por contribuir com o projeto Versin.

Versin é uma plataforma universal de composição musical com integração entre inteligência artificial, identidade criativa, sincronização musical e conexão entre usuários através da música e da expressão artística.

Este documento define as diretrizes para manter o projeto organizado, seguro e consistente.

---

# Objetivo do Projeto

O objetivo do Versin é criar um ecossistema criativo onde:

- usuários possam compor, conectar e compartilhar música,
- a IA auxilie no processo criativo sem substituir a identidade humana,
- o sistema funcione de forma modular, escalável e ética,
- a composição musical seja tratada como identidade digital e expressão pessoal.

# Regras Gerais

## Código Limpo

- utilize nomes claros,
- evite arquivos gigantes,
- prefira modularização,
- mantenha responsabilidade única.

## Segurança

Nunca envie:

- tokens,
- APIs privadas,
- .env,
- credenciais,
- dados sensíveis.

## Ética

Não são aceitos:

- sistemas abusivos,
- rastreamento invasivo,
- conteúdo malicioso,
- manipulação enganosa.

# Pull Requests

Antes de abrir PR:

- teste funcionalidades,
- revise conflitos,
- organize arquivos,
- atualize documentação.

Inclua:

- objetivo,
- mudanças realizadas,
- impacto esperado,
- screenshots se necessário.
