# Hashing e Registro

## Objetivo

Garantir:

- prova de autoria,
- rastreabilidade,
- integridade,
- histórico verificável.

---

# Tecnologias

- SHA-256
- SHA-3
- assinatura criptográfica
- timestamps

---

# Registro de Letras

Usuários podem:

- criar letras,
- importar .txt,
- importar markdown,
- registrar conteúdo manualmente.

Cada conteúdo gera uma hash única.

---

# Registro de Beats

O sistema suporta:

- .mp3
- .wav
- stems
- previews

---

# Fingerprint Sonora

Os arquivos recebem:

- fingerprint sonora,
- hash criptográfica,
- timestamp verificável.

---

# Hash Evolutiva

Toda alteração cria:

```text
v1 → hash A
v2 → hash B
v3 → hash C
```

Isso preserva:

- histórico,
- evolução,
- rastreabilidade.

---

# Prova Jurídica

Os registros podem ser exportados como:

- certificados,
- PDFs,
- provas temporais.
