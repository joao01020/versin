# docs/contracts.md

# Sistema Jurídico

## Objetivo

Criar validação jurídica distribuída baseada em:

- hashes,
- timestamps,
- assinatura digital,
- histórico verificável.

---

# Registro de Conteúdo

Usuários podem registrar:

- letras,
- beats,
- stems,
- projetos,
- contratos.

---

# Estrutura do Documento

O sistema gera PDFs contendo:

- participantes,
- wallets,
- timestamps,
- hashes,
- descrição do projeto,
- validações.

---

# Fluxo de Validação

```text
Documento criado
    ↓
Hash pré-gerada
    ↓
Usuários validam
    ↓
Assinaturas digitais
    ↓
Hash consolidada
```

# Assinatura Distribuída

Todos os participantes precisam aprovar o contrato.

---

# Controle de Percentuais

O projeto não pode ser finalizado sem:

- aprovação coletiva,
- definição de participação,
- consenso entre envolvidos.

---

# Negociação Assistida

A IA pode:

- sugerir equilíbrio,
- detectar abuso,
- evitar travamentos,
- incentivar consenso.
