# docs/notifications.md

# Notificações Inteligentes

## Objetivo

Enviar notificações contextuais baseadas em:

- compatibilidade,
- interesses,
- objetivos,
- comportamento criativo.

---

# Estrutura

Exemplo:

```text
wallet@84XA... possui relação com seus objetivos criativos.
Deseja se conectar?
```

---

# Catálogo Contextual

Ao abrir uma recomendação:

- mostrar apenas projetos relevantes,
- explicar compatibilidade,
- detalhar motivos do match.

---

# Tipos de Notificações

- matches,
- convites,
- validações,
- aprovações,
- revisões,
- entregas,
- negociações.

---

# IA Contextual

A IA evita:

- spam,
- recomendações irrelevantes,
- excesso de notificações.

O objetivo é manter relevância e precisão.
